# 🚀 Quick Installation Guide

## Step 1: Install Extension

1. **Extract Files**
   - Unzip `quantum-adblocker-v5.0.zip` to a folder

2. **Load in Chrome/Edge/Brave**
   - Open browser and go to `chrome://extensions`
   - Enable "Developer mode" (top right)
   - Click "Load unpacked"
   - Select the extracted folder
   - Extension icon appears in toolbar ✓

## Step 2: Configure AI (Optional but Recommended)

### Get API Key (Free)
1. Go to https://openrouter.ai
2. Sign up with Google/GitHub
3. Go to Keys section
4. Create new key
5. Copy the key (starts with `sk-or-v1-...`)

### Add to Extension
1. Click extension icon in toolbar
2. Click "Settings & AI Configuration"
3. Paste API key in "OpenRouter API Key" field
4. Click "Save Settings"
5. Click "Test AI Connection" to verify
6. Done! ✓

## Step 3: Test It

1. Visit any website with ads
2. Watch ads disappear automatically
3. Click extension icon to see statistics
4. Ads blocked, trackers stopped, popups prevented!

## Troubleshooting

**Extension not loading?**
- Make sure all files are extracted
- Check Developer mode is ON
- Try different browser

**Ads still showing?**
- Configure AI detection (Step 2)
- Click "Update Filter Lists" in settings
- Some new ad formats take time to block

**Performance slow?**
- Disable AI temporarily in settings
- Clear AI cache
- Use only OpenRouter (not all 3 APIs)
